# utsupedia
うつ病に関する知識とコラムと相談フォームを作る

## インストール
 
```
$ git clone https://github.com/t1k2a/docker-laravel-handson-MVC.git
$ cd docker-laravel-handson
$ docker-compose up -d --build
$ docker-compose exec app ash
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan migrate
```
